Cascading Style Sheets (CSS) in a style sheet language used for describing the look and formating of a document writen in a markup language. Although most often used to change the style of web pages and user interfaces writen in HTML and XHTML, the language can be applied to any kind of XML document, including plain XML, SVG and XUL.
