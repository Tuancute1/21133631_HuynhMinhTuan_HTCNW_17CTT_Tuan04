Cascading Style Sheets Book

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formating of a document written in a markup language. Although most often used to change the style of web pages and user interfaces written in HTML and XHTML, the language can be applied to any kind of XML document, including plain XML, SVG and XUK. Along with HTML and JavaScript, CSS is a conmerstone technology used by most website to create visually engaging webpages, used interfaces for web applications, and user interfaces for many mobile applications.
